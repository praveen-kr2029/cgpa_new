# 🎓 SRM CGPA Calculator

A simple, responsive, and user-friendly **CGPA Calculator** designed for **SRM IST students**.  
This web app allows users to dynamically add subjects, enter grades and credits, and calculate CGPA instantly.

---

## 🚀 Features

- 📌 Dynamic subject entry (choose number of subjects)
- 🎯 Grade-based calculation (O, A+, A, B+, B, C, F)
- 🧮 Credit-based weighted CGPA calculation
- 💻 Clean & modern UI with glassmorphism design
- ⚡ Instant results without page reload
- 📱 Fully responsive (works on mobile & desktop)

---

## 🛠️ Technologies Used

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**

---

## 📊 Grade Point System Used

| Grade | Points |
|------|--------|
| O    | 10     |
| A+   |  9     |
| A    | 8      |
| B    | 7      |
| B+   | 6      |
| C    | 5      |
| F    | 0      |

---

## 🧑‍💻 How to Use

1. Enter the **number of subjects**
2. Click **Add Subjects**
3. Select the **grade** and enter **credits** for each subject
4. Click **Calculate CGPA**
5. 🎉 View your CGPA instantly
